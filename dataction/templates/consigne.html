consigne d'utilisation :

soumettre une image

1. avant de prendre votre photo vous devez imperativement activer la géolocalisation sur votre appareil ainsi que l'option "Tags de localisation" dans les options de votre appareil photo

2. pour soumettre votre image vous devez la placé dans le répertoire image de votre ordinateur puis la séléctioné via l'interface accessible depuis la page submit
